  「チャート雛形ボタン / buildChartTemplate が存在しないこと」を検証項目に追加。

### (2) AI指示（Markdown指示書）のプレビュー表示を削除

- 削除したのは書き出しパネル下部の textarea（`#exportText`）。`buildMarkdown()` の結果を
  常時表示していたもので、パネル見出しにも「下はAI用指示のプレビュー」とあったため、
  これが「AI指示のプレビュー」に該当すると判断（別candidateはなし）。
- 関連コード（`updateExport()` / `exportDirty` / 入力ハンドラ / `.json-box` CSS）も削除。
  「AI用指示を保存」ボタンは残し、保存時に `buildMarkdown()` を直接呼ぶ形に変更。
- **仕様上の注意**: 旧実装ではプレビュー textarea を手編集すると、その内容がそのまま
  保存される隠し挙動がありました。プレビュー削除に伴いこの挙動もなくなり、
  保存されるのは常に生成された Markdown になります（通常運用では影響なし）。

### (3) 使い方をガイドツアー（チュートリアル形式）に強化

- 使い方モーダルに「**ガイドツアーで始める**」ボタンを追加。実画面上で「次に押すボタン」を
  スポットライト（周囲を暗転）+ 吹き出しで順番に案内する5ステップ:
  デモ音源読み込み → 再生 → 打刻（キー追加 / K） → 微調整（±ms / 拍へ） → AI用指示を保存。
- 各ステップは実際に操作すると自動で次へ進む（250ms ポーリングで達成判定）。
  「次へ」で読み飛ばし可、「終了」/ `Esc` でいつでも中断可。
- 中断中は進捗（再生済み・打刻済み等のフラグ）を保持し、`?` →「ガイドツアーで始める」で
  達成済みステップを飛ばして続きから再開。完走後は次回まっさらから開始。
- README に「ガイドツアー」節と `images/screenshot-tour.png` を追加。
